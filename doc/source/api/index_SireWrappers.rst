BioSimSpace._SireWrappers
=========================

The *_SireWrappers* package contains a set of thin wrappers around
`Sire <https://github.com/michellab/Sire>`_ objects. These are internal
BioSimSpace types and aren't typically instantiated directly by the
user, instead being created indirectly by BioSimSpace, e.g. through .
:class:`BioSimSpace.IO.readMolecules <BioSimSpace.IO.readMolecules>`.
