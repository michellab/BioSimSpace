BioSimSpace.Protocol
====================

The *Protocol* package contains functionality for defining generic molecular
dynamics protocols. The protocols contain configurable options that are
common across different molecular dynamics packages. In combination with a
molecular :class:`System <BioSimSpace._SireWrappers.System>`, protocols can
be used to create a molecular dynamics :class:`Process <BioSimSpace.Process>`.
