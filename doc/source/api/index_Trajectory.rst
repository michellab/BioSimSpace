BioSimSpace.Trajectory
======================

The *Trajectory* package contains tools for reading and analysing molecular
dynamics trajectories. Trajectory files can be read directly, or a
:class:`Trajectory <BioSimSpace.Trajectory.Trajectory>` object can be bound
to a :class:`Process <BioSimSpace.Process>` to extract and analyse frames
from a running simulation. We currently support `MDAnalysis <https://www.mdanalysis.org>`__,
`MDTraj <https://www.mdtraj.org>`__, and `Sire <https://sire.openbiosim.org>`__ as
backends.
